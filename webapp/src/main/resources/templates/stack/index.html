About the project and the stack

Pizza chain website with Rest API.

Written back-end of the site, web representation and implemented REST API.

Part of the site is available to all users. More functionality is available only to registered and authorized users. Site administrator can modify and add all entities: Create new pizzas, new ingredients, new cafes, add or remove pizzas to the caf√© menu. Add or remove ingredients for a pizza. Check and set pizza prices. The ingredients are differentiated by type (Vegetables, seafood etc.). The administrator can add and change types. The pizza has different bases and sizes (thin dough, classic dough, small, large, etc.).
